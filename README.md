# Sales Forecasting and Business Performance Dashboard

## Project Description
This project analyzes historical sales data and predicts future sales trends using forecasting techniques. It helps understand business performance through charts and data visualization.

## Features
- Sales Data Analysis
- Monthly Sales Trend
- Sales Forecasting
- Business Performance Analysis
- Data Visualization using Graphs

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- CSV Dataset

## Project Files
- sales_forecasting_dataset.csv
- sales_forecasting.ipynb
- dashboard.png
- screenshots

## Output
This project generates:
- Sales trend graph
- Forecasted sales data
- Business performance analysis
